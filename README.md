# Case File #371 — The Blackwood Earrings

Desktop-first, single-page interactive noir detective experience built with vanilla HTML, CSS, and JavaScript. It is designed as a cinematic portfolio website with environmental storytelling, not as a conventional game.

## Run locally

Open `index.html` directly, or serve this folder with any static server:

```powershell
python -m http.server 4317
```

Then open `http://127.0.0.1:4317`.

## Narrative interaction path

1. Open the folder.
2. Read the report and optionally enlarge either photograph.
3. Close the file.
4. Turn over any three suspect photographs.
5. Move the coffee cup.
6. Choose the thief.
7. Restart the case from the final report.

All generated imagery used by the page is stored under `assets/`.
